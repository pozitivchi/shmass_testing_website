# Веб-система тестирования команды ШМАСС

Профессиональная платформа для проведения тестирования, построенная на PHP. Система поддерживает гибкую настройку баллов, множественный выбор ответов и экспорт результатов.

---

## 🌟 Основные возможности

* **Каталог тестов:** Выбор и прохождение доступных тестов на главной странице.
* **Гибкая логика вопросов:** * Настраиваемая «стоимость» каждого вопроса в баллах.
    * Поддержка выбора нескольких вариантов ответа одновременно.
* **Работа с результатами:** * Мгновенный вывод итогов на экран.
    * Возможность скачивания результата в формате `json`.
* **Личный кабинет:** Зарегистрированные пользователи могут просмотреть результат своего последнего пройденного теста.
* **Админ-панель:** Полный цикл CRUD для управления тестами (доступно пользователям с ролью администратора).

---

## 🛠 Технологический стек

* **Язык программирования:** PHP 8.2+
* **База данных:** MySQL (хранение данных пользователей и прав доступа).
* **Хранение тестов:** JSON-файлы (в директории `/data`).
* **Среда разработки/запуска:** XAMPP (Apache + MySQL).

---

## ⚙️ Установка и запуск

### 1. Установка Portable версии
1. Скачайте zip архив со вкладки realese
2. Распакуйте его по короткому пути без кириллицы (рекомендуется корень диска, например `C:\xampp`).
3. Запустите **XAMPP Control Panel**.
4. Активируйте модули **Apache** и **MySQL**, нажав кнопку **Start**.
5. Cайт будет доступен по адресу:  
   `http://127.0.0.1/shmass_testing_website/index.php`

### 2. Развертывание с формированием окружения вручную
1. Скачайте и установите последнюю версию [XAMPP](https://www.apachefriends.org/index.html).
2. Поместите папку с исходным кодом проекта в директорию `htdocs` вашего XAMPP (например, `C:\xampp\htdocs\shmass_testing_website\`).
3. Запустите **XAMPP Control Panel**.
4. Активируйте модули **Apache** и **MySQL**, нажав кнопку **Start**.
5. Импортируйте базу данных из дампа schema.sql через **phpMyAdmin** (обычно доступен по адресу `http://localhost/phpmyadmin`).
6. Cайт будет доступен по адресу:  
   `http://127.0.0.1/shmass_testing_website/index.php`

## 🔑 Роли и доступ

* **Пользователь:** Проходит тесты, сохраняет результаты.
* **Администратор:** Имеет доступ к управлению контентом тестов.
    * *Важно:* Роль администратора назначается вручную разработчиком через изменение соответствующего поля в базе данных MySQL.

---

## 📂 Структура данных

Все тестовые задания хранятся в папке `data/` в формате JSON. Это позволяет быстро редактировать контент без прямого обращения к SQL-таблицам.

Пример структуры вопроса:
```json
{
    "category": "Литература",
    "questions": [
        {
            "points": "10",
            "question": "Кто из этих писателей писал сказки?",
            "correct_indices": [
                "0",
                "1"
            ],
            "options": [
                "Александр Пушкин",
                "Ганс Христиан Андерсен",
                "Лев Толстой"
            ]
        }
}
```
